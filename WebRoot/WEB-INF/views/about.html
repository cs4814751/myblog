一个脱离了高级趣味的人

我是山东大学控制工程专业的一名在校研究生，今年研二，曾经学过嵌入式，后来走上了Java后端这条不归路。
这个网站，当然，也是一个脱离了高级趣味的博客。
相对于微博，朋友圈，我在这里可以更加自由的说说话，不用顾忌任何人的感受，反正也没几个人会看到。不像其他的社交媒体，想说的话，想发的文章，多少次欲言又止，充满顾忌。这个博客，反正我就随便说说，拿来当技术的试验田，也挺好的。
整个博客的选用的开源框架如下：

  • Spring MVC
  • Spring Security
  • Spring Data Redis
  • Mybatis
  • Redis
  • PageHelper
  • Marked.js
  • SimpleMDE
  • NProgress
  • Materializecss

实现了博客的日常所需要的所有功能，Redis提供缓存，实现了博客访问次数实时统计，数据库选用MySql进行持久化存储。
接下来还将对本博客进行持续性的优化和重构。
